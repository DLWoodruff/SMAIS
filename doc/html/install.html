Installation and Quick Start¶

This is a terminal application.

Installation¶

  1. Verify that a Python version 3.8 or higher is installed.

  2. Verify that git is installed

  3. Install a solver such as cplex, glpk, gurobi, ipopt or etc. so that is can be run from the command line.

  4. Install Pyomo.

  5. Install mpi-sppy using a github clone.

  6. Install boot-sp using a github clone.

  7. cd to the directory where you want to put smais and give these commands:

$ git clone https://github.com/DLWoodruff/SMAIS
$ cd SMAIS
$ pip install -e .

For parallel operation, you will need to install multiprocessing_on_dill <https://pypi.org/project/multiprocessing_on_dill/>

Quick Start¶

To test your installation, assuming that you have Gurobi installed,

$ cd examples
$ bash newsvendor.bash

If you don’t have Gurobi Installed, edit examples/newsvendor_is.json and change the solver name.

smais
